# Search properties

Retuns a list of properties that match the search criteria

## Header

```
Content-Type: application/x-www-form-urlencoded

```

`GET`

```- copy
/Property?name=kea&country=IS
```

## Parameter

| **Field** | **Type** | **Description**                                                                      |
| :-------- | :------- | :----------------------------------------------------------------------------------- |
| area      | String   | the search area e.g. &attiki                                                         |
| city      | String   | the search city e.g. &athens                                                         |
| name      | String   | The hotel name. all hotels that contain that name will be in the list e.g. &name=kea |
| hotel     | String   | ISO 3166-1-alpha-2 country code e.g. &country=GR                                     |

## Success Response (200)

| **Field** | **Type**               | **Description** |
| :-------- | :--------------------- | :-------------- |
| hotels    | array of hotel objects | the hotel list  |
