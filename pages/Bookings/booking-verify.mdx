# Get reservation information

In order to check if a booking has been submitted successfully, API offers the possibility to immediately verify the booking status once the booking
was submitted by sending an additional "/booking_verify" request.

## Header

```
Content-Type: application/x-www-form-urlencoded
```

`GET`

```- copy
/booking_verify/:reservation_id

```

## Parameter

| **Field**      | **Type** | **Description**     |
| :------------- | :------- | :------------------ |
| reservation_id | String   | The reservation ID. |
